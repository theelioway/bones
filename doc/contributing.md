# Contributing

```shell
git clone https://gitlab.com/elioway/elioway.gitlab.io.git elioway
cd elioway
git clone https://gitlab.com/elioway/eliobones.git
cd eliobones
git clone https://gitlab.com/eliobones/bones.git
cd bones
npm i|yarn
npm|yarn devstart
```
